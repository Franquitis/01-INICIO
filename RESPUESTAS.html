¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML

- HTML significa HyperText Markup Language o lenguaje de marcado de hipertexto. Se dice que es un lenguaje de marcas ya que los elementos del codigo se representan en etiquetas. 1- <h1> 2- <p> 3- <title> 4- <head> 5- <h2> 6- <h3> 7- <h4> 8- <h5>.

¿Qué es y que función cumple el CSS?

el CSS es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc. Haciendo que de esta manera sean mas atractivas las paginas Web.

¿Cómo se incorpora el CSS dentro del HTML?

Para vincular styles.css a index.html, añade la siguiente línea en algún lugar dentro del <head> del documento HTML:

<link rel="stylesheet" href="styles.css">
